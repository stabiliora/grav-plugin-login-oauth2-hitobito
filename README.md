# Hitobito OAuth2 Plugin for Grav CMS

This plugin provides an extra authentication provider [Hitobito](https://github.com/hitobito/hitobito/blob/master/doc/developer/people/oauth.md) not included in the [Login OAuth2 Plugin](http://github.com/trilbymedia/grav-plugin-login-oauth2) for [Grav CMS](http://github.com/getgrav/grav).

**NOTE:** Please use the same **Callback URIs** and configuration from **Login OAuth2 Plugin**.
